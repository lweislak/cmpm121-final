Tools and Materials
--------------------------------------------
Our team will be using Unity for the final game. We both have experience in Unity and using GitHub version control. 
We will be exporting the project to be played on both Mac and Windows.

Unity utilizes C# so that is what we will be working in. Visual Studio and Visual Studio Code will be the primary IDEs.

We will be importing free, premade assets from the Unity Asset Store and creating the map with the built in Unity tile editor.
It is easier to create the maps in Unity instead of importing them and there is still plently of custom tile sets avaliable,
which is why we are using the built in editor.

Phaser is our alternate engine choice; Unreal Engine is not good for 2D and we both have knowledge of Phaser.

Outlook
--------------------------------------------
Our team is hoping to implement particle effects when sowing and reaping plants to add more "juice" to the game.

The visual design is going to be the most difficult part of the project. This includes making the visuals look cohesive.

We hope to gain more knowledge about implementing game systems on a short time frame.
We don't have to come up with the premise, so it'll be a good test of how to make a game based on given guidelines.
